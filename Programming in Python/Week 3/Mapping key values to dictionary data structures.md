### Question 1:

What will be the output of the following code:
```
a = [[96], [69]]

print(''.join(list(map(str, a))))
```
Answer: **“[96][69]”**

### Question 3:

What will be the output of the following code:
```
z = ["alpha","bravo","charlie"]
new_z = [i[0]*2for i in z]
print(new_z)
```
Answer: **['aa', 'bb', 'cc']**
